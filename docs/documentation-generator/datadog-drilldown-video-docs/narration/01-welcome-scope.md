This walkthrough covers the Datadog Drilldown project: A I assisted root cause analysis for production incidents.

We combine Datadog telemetry, Embabel style agents, and DICE — domain integrated context engineering — for durable incident memory.

The optional Q U B O path is not a separate product silo. A report enricher in the R C A agent ties DICE’s proposition side and numerical optimization together on the same incident report, so teams get extracted facts from text and, when the case is complex enough, a jointly consistent read on which root cause hypotheses belong in the short list.
